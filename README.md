# Auto Wizard

A vehicle recommendation website that helps users find their perfect car through a 13-question assessment.

## Features

- 13-question assessment covering lifestyle, budget, and preferences
- 16 vehicle categories with 200+ model recommendations
- Expert consultation booking system
- Responsive design with modern UI

## Deploy to Vercel

1. Upload all these files to a GitHub repository
2. Go to vercel.com and import the repository
3. Vercel will automatically detect Next.js and deploy

Your site will be live in about 2 minutes!
